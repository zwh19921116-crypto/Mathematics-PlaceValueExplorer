# Place Value Explorer

A professional and engaging HTML app for primary school learners to understand place value.

## Features

- Number explorer from 0 to 999,999
- Place value cards for each digit position
- Expanded form display
- Number in words
- Visual block representation of each place
- Interactive quiz with score tracking
- Responsive design for desktop and mobile

## Run Locally

1. Open the project folder in VS Code.
2. Open `index.html` directly in a browser, or use VS Code Live Server.

## Deploy To GitHub Pages

1. Push the repository to GitHub.
2. In repository settings, open **Pages**.
3. Set source to deploy from branch `main` and folder `/ (root)`.
4. Save and wait for deployment.

Your app will be available on the GitHub Pages URL generated by GitHub.
